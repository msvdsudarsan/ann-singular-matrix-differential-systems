# Results Directory

This directory stores numerical outputs generated by the MATLAB PINN experiments.

Contents may include:
- Saved solution trajectories
- Error metrics (MAE, max error, loss values)
- Intermediate convergence data

All reported numerical values in the accompanying manuscript were obtained
from executions of the scripts provided in the `matlab/` directory.

Users may regenerate these results by running:

```matlab
run_all_experiments
